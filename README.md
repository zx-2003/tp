# TutorMap
![Ui](docs/images/Ui.png)

**TutorMap** is a desktop application for managing your contact details as a private tutor. While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

---
As a private tutor, **TutorMap** provides an easy way to:
- Categorise contacts by type: `students`, `parents`, `agents`
- Tag the subjects taught for different students
- Link and visualise the relationships between contacts
